KnowYourCA

KnowYourCA is a full-stack MERN application designed to connect users with verified Chartered Accountants through a modern, secure, and real-time platform.
It enables users to discover CAs based on specialization and location, view professional portfolios, leave reviews, and communicate instantly through a gated subscription-based chat system.
Built with a scalable architecture and premium UI design, KnowYourCA delivers a commercial-grade experience.
